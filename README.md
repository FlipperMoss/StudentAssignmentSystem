# Student Assignment Management System

A web-based assignment management platform built using PHP and MySQL.

## Features Implemented

- User Registration
- User Login
- Password Hashing
- Session Management
- Student Dashboard
- Lecturer Dashboard
- Role-Based Access Control

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- XAMPP
- Git/GitHub

## Future Features

- Assignment Creation
- Assignment Submission
- File Uploads
- Marks and Feedback
- Dashboard Statistics
